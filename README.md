# TDD React

Done following the tutorial found at [testdriven.io](https://testdriven.io/blog/tdd-with-react-jest-and-enzyme-part-one/)

## Local Setup
```sh
$ git clone git@github.com:testdrivenio/react-calculator.git
```

```sh
$ cd react-calculator
```

```sh
$ npm i
```

## Run Locally
```sh
$ npm start
```

## Run Tests
```sh
$ npm test
```
